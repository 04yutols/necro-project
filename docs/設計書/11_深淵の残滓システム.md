# 11 — 深淵の残滓システム設計書

> 最終更新: 2026-05-06  
> 実装/予定ファイル: `src/logic/ResidueScore.ts`, `src/components/legion/LegionHub.tsx`, `src/logic/ResidueSystem.ts`（予定）, `src/services/ResidueService.ts`（予定）

本システムは、プレイヤーの長期的なエンゲージメント（プレイ継続率）を最大化するため、
**意図的な確率の偏重（Weighting）** と **心理学的な報酬系アルゴリズム** を実装する。

---

## 1. システム概要

「深淵の残滓」は崩壊スターレイルの「遺物（Relic）」に相当するキャラクター強化システム。
5つの部位スロットに装備し、メインステータス＋サブステータスで性能が決まる。
Lv1〜20まで強化でき、4上がるごとにサブステータスが1つ成長する（Lv20まで計5回）。

## 2. 部位別メインステータス排出確率（The Unfair Distribution）

プレイヤーの火力に直結するステータスの排出率を意図的に絞ることで「**厳選の沼**」を形成する。

| 部位 | メインステータス候補 | 内部確率 |
|---|---|---|
| **head** | HP（実数値） | **100%（固定）** |
| **arms** | ATK（実数値） | **100%（固定）** |
| **chest** | HP% | 26% |
| | ATK% | 26% |
| | DEF% | 26% |
| | ソウルゲージ回復効率 | 10% |
| | 魔神化持続力 | 10% |
| | *(確率計)* | *(98% ※端数調整)* |
| **waist** | HP% | 19% |
| | ATK% | 19% |
| | DEF% | 19% |
| | 属性DMGバフ（FIRE） | 5% |
| | 属性DMGバフ（ICE） | 5% |
| | 属性DMGバフ（THUNDER） | 5% |
| | 属性DMGバフ（DARK） | 5% |
| | 属性DMGバフ（LIGHT） | 5% |
| | 状態異常命中 | 2.5% |
| | *(確率計)* | *(≒84.5% ※残余は将来追加枠)* |
| **legs** | HP% | 22% |
| | ATK% | 22% |
| | DEF% | 22% |
| | 会心率（critRate） | 10% |
| | 会心ダメージ（critDmg） | 10% |
| | 与治癒効果 | 10% |
| | 状態異常耐性 | 4% |
| | *(確率計)* | **100%** |

> **設計意図**: プレイヤーが最も欲する「属性バフ腰（各5%）」と「会心足（各10%）」を
> 意図的に低確率に設定することで、妥協枠（オフセット採用）の戦略的判断を生み、
> 日々の周回モチベーションを長期にわたって担保する。

## 3. サブステータス付与・強化アルゴリズム（The 4-Layer RNG）

最高レアリティ（Epic / Legendary）の残滓では、以下4層の乱数が品質を決定する。

---

### Layer 1 — 初期サブステータス付与（重み付き抽選）

メインステータス決定後、重複しないサブステータスが **3〜4個** 付与される。
抽選は均等ではなく、以下の重みに基づく加重抽選式を使用する:

$$P(i) = \frac{W_i}{\sum_{j} W_j}$$

| Weight | 対象ステータス |
|---|---|
| **6（高）** | HP実数、ATK実数、DEF実数 |
| **4（中）** | HP%、ATK%、DEF%、ソウル回復効率 |
| **3（低）** | 会心率（critRate）、会心ダメージ（critDmg） |

- メインステータスと同種のサブは抽選プールから除外
- **Epic**: サブ3個固定でスタート
- **Legendary**: サブ4個固定でスタート

```typescript
// 将来実装イメージ (src/logic/ResidueSystem.ts)
const SUB_STAT_WEIGHTS: Record<string, number> = {
  hpFlat:  6, atkFlat: 6, defFlat: 6,
  hpPct:   4, atkPct:  4, defPct:  4, soulRegen: 4,
  critRate: 3, critDmg: 3,
};

function rollSubStats(mainStatType: string, count: number): string[] {
  const pool = Object.entries(SUB_STAT_WEIGHTS)
    .filter(([key]) => key !== mainStatType);
  // 重み付き非復元抽選でcount個選出
  return weightedSampleWithoutReplacement(pool, count);
}
```

---

### Layer 2 — 強化時の選出確率（The 25% Trap）

**Lv+4ごと（Lv4/8/12/16/20の計5回）** に、既存サブステータスから1つを選出して強化する。

> **仕様**: 強化対象の選出確率は、ステータスの種類に関わらず **完全均等（各25%）**。

| サブ構成例 | 確率配分 |
|---|---|
| 会心率 / 会心ダメ / ATK% / DEF実数 | 各 25% |

> **UX効果**: 初期サブが「会心率・会心ダメ・ATK%・DEF実数」という理想配置でも、
> 毎回25%の確率でDEF実数が選ばれ続けるリスクが存在する。
> この **「絶望」** の可能性があるからこそ、理想強化成功時の **「カタルシス」** が最大化される。

---

### Layer 3 — 伸び幅のTierシステム

選ばれたサブステータスの上昇値は固定ではなく、**4段階のTierから均等（各25%）** に抽選される。

| Tier | 最大値比率 | 選出確率 |
|---|---|---|
| Tier 1 | 70% | 25% |
| Tier 2 | 80% | 25% |
| Tier 3 | 90% | 25% |
| Tier 4 | 100% | 25% |

> Lv20まで5回強化した場合の期待値は最大値の **85%**（(70+80+90+100)/4）。
> 全Tier4（最大値×5回）の確率は **0.25⁵ ≈ 0.098%**（千回に1本の「神器」）。

---

### Layer 4 — 初期サブ数の揺らぎ（Epicのみ適用）

| レアリティ | Lv0時点のサブ数 | 備考 |
|---|---|---|
| RARE | 2個 | 強化でのみ増加 |
| EPIC | **3または4個** | 4個スタートは「当たり引き」 |
| LEGENDARY | 4個固定 | Layer4は非適用 |

Epic の初期4個スタート確率は **20%**（設定値。後日調整可）。
4個スタート品は強化不要サブが多いため、市場価値が高い。

## 5. ゲームループとUXのコントロール（Retention Strategy）

### A. スタミナシステム（ペース配分装置）

残滓をドロップする「**深淵の迷宮（ドメイン）**」のクリア報酬受け取りには、
時間回復型スタミナ「**ネクロフォース**」の消費を要求する。

| パラメータ | 値（暫定） |
|---|---|
| 1回入場コスト | ネクロフォース × 20 |
| 自然回復速度 | 1個 / 6分（1日240個） |
| 1日周回上限目安 | 12回 |

> **目的**: プレイヤーが数日で理想装備を完成させて離脱する「燃え尽き」を防ぎ、
> 厳選を数ヶ月単位の **「日課（ルーティン）」** へと変容させる。

---

### B. 残滓スコア（ソーシャル誘発システム）

残滓の「実戦的価値」をスカラー値に集約し、UIに可視化する。

```
残滓スコア = critRate × 2 + critDmg + atk% × 1.5 + hp%
```

| 評価 | スコア目安 |
|---|---|
| C（普通） | 0〜19 |
| B（良品） | 20〜34 |
| A（優品） | 35〜49 |
| S（稀少） | 50〜64 |
| SS（神器） | 65〜 |

> **目的**: SNS・Discord で「SS出た！」「また防御しか伸びなかった…」という
> 文化を自然発生させ、コミュニティの力で周回意欲を **再生産** させる。
> スコア計算は将来 `src/logic/ResidueScore.ts` に実装予定。

---

### C. 魂の錬成（Soul Transmutation）— 救済システム

完全なランダム厳選による「**詰み**」や「**虚無感**」を軽減する救済システム。

**フロー**:
```
不要な Epic/Legendary 残滓を砕く
  → 錬成ポイント獲得
      ├ Epic:     30pt
      └ Legendary: 80pt

錬成ポイントを消費して「特注残滓製造権」を入手:
  └ 部位・メインステータス・サブ2種を指定可能

コスト（製造権1枚あたり）:
  head / arms: 600pt
  chest:       800pt
  legs:        1000pt
  waist:       ★2500pt（最高コスト）
```

> **設計意図**:
> - ハズレ品に「錬成ポイントの餌」としての価値を持たせる
> - 腰部位を高コスト設定することで、自力厳選の達成感を維持しつつ救済を提供
> - サブ2種の指定は「方向性の保証」であり全サブの指定ではない（運の余地を残す）
