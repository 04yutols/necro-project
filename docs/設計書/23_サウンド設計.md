## 1. 設計哲学 — 原神 / Star Rail レベルのサウンド UX 基準

### 1.1 4つのサウンド原則

| 原則 | 原神 / Star Rail の実現例 | 本作への適用 |
|---|---|---|
| **音が感情を先導する** | 5★バースト発動時に「世界が止まる」ような一瞬の無音→爆発SE | 魔神化発動で通常BGMが0.2秒フェードアウト→深淵のドローン→魔神化BGMレイヤー追加 |
| **動的な音楽レイヤー** | 探索時は静かなオーケストラ、戦闘開始でパーカッション・レイヤーが積み重なる | HOME環境音→MAP探索音楽→BATTLE通常→BATTLE BOSS→魔神化オーバーレイ の5層設計 |
| **SEが世界観を語る** | キャラクターごとに固有の召喚SE・スキルSE（神里綾華の氷音、モナの水音） | 属性別スキルSE・職業別魔神化SE・種族別モンスター召喚SEで「世界に生きている音」を演出 |
| **沈黙を恐れない** | SSRドロップ演出の0.8秒の暗転・無音がプレイヤーの期待を最大化する | SSRドロップ: VFX Phase 3（黒帯暗転）に完全無音 → 一転してオーケストラ爆発 |

### 1.2 Gothic-Morphism サウンドパレット

本作のサウンドアイデンティティは **ゴシック建築 × 死霊術 × 勇者の遺産の葛藤** を音で表現する。

| 要素 | 音響テクスチャ | 具体音源 |
|---|---|---|
| 主人公の勇者側 | 明るい金管・弦楽、英雄的ファンファーレ | 職業スキルSE・勝利ジングル |
| 主人公の魔族側 | 低音ドローン、逆再生成分、金属共鳴 | 魔神化SE・死霊術SE |
| 世界観・環境 | パイプオルガン・石の共鳴・骨の打音 | UI SE・ロビーBGM・ダンジョン環境音 |
| ドロップ・高揚感 | 上昇する弦楽ハーモニクス・Void Purple の「光音」 | SSR/UR ドロップ演出SE |

### 1.3 技術方針

**現行**: `useGothicSound.ts` が Web Audio API 合成（オシレーター＋ノイズバッファ）でUIサウンドを実装済み。  
**拡張方針**: 以下の2トラックを並行する。

| トラック | 対象 | 手法 |
|---|---|---|
| **合成SE** | UI操作・攻撃ヒット・状態異常ティック等の短い即時SE | `AudioContext` オシレーター合成（ファイル不要） |
| **ファイルBGM** | BGM・長尺の魔神化ループ・SSRドロップ演出BGM | `<audio>` タグ または `AudioBufferSourceNode`（mp3/ogg） |

### 2.2 BGM 遷移ルール

```
HOME_LOBBY
  │  「出撃・マップ」タップ
  ↓  クロスフェード 1.5s
MAP_EXPLORE
  │  ステージ選択（通常ステージ）
  ↓  クロスフェード 0.8s
BATTLE_NORMAL
  │  BOSSステージ OR WAVE3到達
  ↓  クロスフェード 0.5s（緊張感を高速で切り替え）
BATTLE_BOSS
  │  魔神化発動
  ↓  BATTLE_BOSS を 0.3s で -12dB に絞りながら DEMON_OVERLAY 再生開始（重畳）
BATTLE_BOSS + DEMON_OVERLAY（並行再生）
  │  魔神化終了
  ↓  DEMON_OVERLAY を 1s でフェードアウト、BATTLE_BOSS を元の音量に戻す
  │  バトル終了（勝利）
  ↓  即時カット → RESULT_WIN → 0.5s 後 MAP_EXPLORE or HOME_LOBBY
  │  バトル終了（敗北）
  ↓  即時カット → RESULT_LOSE → 1s 後 MAP_EXPLORE
```

ストーリーシーン発火時: `StoryOrchestrator` が `useBGM.setScene('STORY_EMOTIONAL' | 'STORY_NEUTRAL')` を呼び出す。シーン終了後は直前のシーンに戻る。

## 4. BGM 管理実装

### 4.1 `AudioService.ts` — シングルトン

```typescript
// src/services/AudioService.ts

export type BGMScene =
  | 'HOME_LOBBY' | 'MAP_EXPLORE' | 'BATTLE_NORMAL' | 'BATTLE_BOSS'
  | 'DEMON_OVERLAY' | 'RESULT_WIN' | 'RESULT_LOSE'
  | 'STORY_EMOTIONAL' | 'STORY_NEUTRAL' | 'NECRO_LAB'
  | 'SSR_REVEAL' | 'UR_REVEAL';

interface BGMChannel {
  audio: HTMLAudioElement;
  gainNode: GainNode;
}

class AudioServiceImpl {
  private static instance: AudioServiceImpl;
  private ctx: AudioContext | null = null;
  private channels: Map<BGMScene, BGMChannel> = new Map();
  private activeScene: BGMScene | null = null;
  private overlayScene: BGMScene | null = null;  // DEMON_OVERLAY 専用

  // BGM ボリューム（0〜1）
  bgmVolume = 0.7;
  seVolume = 0.8;
  muted = false;

  static getInstance(): AudioServiceImpl {
    if (!AudioServiceImpl.instance) {
      AudioServiceImpl.instance = new AudioServiceImpl();
    }
    return AudioServiceImpl.instance;
  }

  private getCtx(): AudioContext | null {
    if (typeof window === 'undefined') return null;
    if (!this.ctx) {
      try {
        this.ctx = new (window.AudioContext || (window as any).webkitAudioContext)();
      } catch { return null; }
    }
    if (this.ctx.state === 'suspended') this.ctx.resume();
    return this.ctx;
  }

  async transitionTo(scene: BGMScene, fadeDuration = 1.0): Promise<void> {
    if (scene === this.activeScene) return;
    const ctx = this.getCtx();
    if (!ctx || this.muted) return;

    // 現在チャンネルをフェードアウト
    if (this.activeScene) {
      const prev = this.channels.get(this.activeScene);
      if (prev) this.fadeGain(prev.gainNode, 0, fadeDuration, ctx);
    }

    // 新チャンネルをフェードイン
    const next = await this.getOrCreateChannel(scene, ctx);
    next.audio.currentTime = 0;
    next.audio.play().catch(() => {});
    this.fadeGain(next.gainNode, this.bgmVolume, fadeDuration, ctx);

    this.activeScene = scene;
  }

  async startOverlay(scene: BGMScene): Promise<void> {
    const ctx = this.getCtx();
    if (!ctx || this.muted) return;
    const ch = await this.getOrCreateChannel(scene, ctx);
    ch.audio.currentTime = 0;
    ch.audio.play().catch(() => {});
    this.fadeGain(ch.gainNode, this.bgmVolume * 0.8, 0.3, ctx);
    this.overlayScene = scene;

    // メインBGMを -12dB に絞る
    if (this.activeScene) {
      const main = this.channels.get(this.activeScene);
      if (main) this.fadeGain(main.gainNode, this.bgmVolume * 0.25, 0.3, ctx);
    }
  }

  stopOverlay(fadeDuration = 1.0): void {
    const ctx = this.getCtx();
    if (!ctx || !this.overlayScene) return;
    const ch = this.channels.get(this.overlayScene);
    if (ch) this.fadeGain(ch.gainNode, 0, fadeDuration, ctx);

    // メインBGMを元の音量に戻す
    if (this.activeScene) {
      const main = this.channels.get(this.activeScene);
      if (main) this.fadeGain(main.gainNode, this.bgmVolume, fadeDuration, ctx);
    }
    this.overlayScene = null;
  }

  private fadeGain(gain: GainNode, target: number, duration: number, ctx: AudioContext): void {
    gain.gain.cancelScheduledValues(ctx.currentTime);
    gain.gain.setValueAtTime(gain.gain.value, ctx.currentTime);
    gain.gain.linearRampToValueAtTime(target, ctx.currentTime + duration);
  }

  private async getOrCreateChannel(scene: BGMScene, ctx: AudioContext): Promise<BGMChannel> {
    if (this.channels.has(scene)) return this.channels.get(scene)!;
    const audio = new Audio(`/audio/bgm/${scene.toLowerCase()}.mp3`);
    audio.loop = !['RESULT_WIN', 'RESULT_LOSE', 'SSR_REVEAL', 'UR_REVEAL'].includes(scene);
    audio.volume = 0;
    const source = ctx.createMediaElementSource(audio);
    const gain = ctx.createGain();
    gain.gain.value = 0;
    source.connect(gain);
    gain.connect(ctx.destination);
    const ch: BGMChannel = { audio, gainNode: gain };
    this.channels.set(scene, ch);
    return ch;
  }

  setVolume(bgm: number, se: number): void {
    this.bgmVolume = bgm;
    this.seVolume = se;
    if (this.activeScene) {
      const ch = this.channels.get(this.activeScene);
      if (ch) ch.gainNode.gain.value = bgm;
    }
  }

  setMuted(muted: boolean): void {
    this.muted = muted;
    this.channels.forEach(ch => {
      ch.gainNode.gain.value = muted ? 0 : this.bgmVolume;
    });
  }
}

export const AudioService = AudioServiceImpl.getInstance();
```

```typescript
// src/hooks/useSoundEffects.ts
import { useCallback } from 'react';
import { AudioService } from '../services/AudioService';
import { useAudioStore } from '../store/useAudioStore';

// Web Audio API合成ヘルパー
function synth(
  ctx: AudioContext,
  type: OscillatorType,
  freqStart: number,
  freqEnd: number,
  duration: number,
  volume: number,
  delay = 0,
) {
  const now = ctx.currentTime + delay;
  const osc = ctx.createOscillator();
  osc.type = type;
  osc.frequency.setValueAtTime(freqStart, now);
  if (freqEnd !== freqStart) {
    osc.frequency.exponentialRampToValueAtTime(freqEnd, now + duration);
  }
  const gain = ctx.createGain();
  gain.gain.setValueAtTime(volume, now);
  gain.gain.exponentialRampToValueAtTime(0.001, now + duration);
  osc.connect(gain);
  gain.connect(ctx.destination);
  osc.start(now);
  osc.stop(now + duration + 0.01);
}

function noise(ctx: AudioContext, duration: number, volume: number, filterFreq = 1000, delay = 0) {
  const now = ctx.currentTime + delay;
  const bufSize = Math.ceil(ctx.sampleRate * duration);
  const buf = ctx.createBuffer(1, bufSize, ctx.sampleRate);
  const data = buf.getChannelData(0);
  for (let i = 0; i < bufSize; i++) data[i] = (Math.random() * 2 - 1);
  const src = ctx.createBufferSource();
  src.buffer = buf;
  const flt = ctx.createBiquadFilter();
  flt.type = 'lowpass';
  flt.frequency.value = filterFreq;
  const gain = ctx.createGain();
  gain.gain.setValueAtTime(volume, now);
  gain.gain.exponentialRampToValueAtTime(0.001, now + duration);
  src.connect(flt);
  flt.connect(gain);
  gain.connect(ctx.destination);
  src.start(now);
  src.stop(now + duration + 0.01);
}

export function useSoundEffects() {
  const seVolume = useAudioStore(s => s.seVolume);
  const muted = useAudioStore(s => s.muted);
  const vol = useCallback((base: number) => muted ? 0 : base * seVolume, [seVolume, muted]);

  const getCtx = useCallback((): AudioContext | null => {
    // AudioService 経由で共有コンテキストを取得
    return (AudioService as any).getCtx?.() ?? null;
  }, []);

  return {
    attackPhysical: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;
      synth(ctx, 'sawtooth', 120, 60, 0.15, vol(0.2));
      noise(ctx, 0.1, vol(0.12), 400);
      navigator.vibrate?.(12);
    }, [vol, getCtx]),

    attackCritical: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;
      synth(ctx, 'sawtooth', 120, 60, 0.15, vol(0.3));
      noise(ctx, 0.1, vol(0.18), 400);
      synth(ctx, 'sine', 880, 880, 0.3, vol(0.12));
      navigator.vibrate?.([15, 20, 15]);
    }, [vol, getCtx]),

    attackWeakness: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;
      synth(ctx, 'sawtooth', 120, 60, 0.15, vol(0.25));
      noise(ctx, 0.1, vol(0.15), 400);
      synth(ctx, 'sine', 660, 880, 0.2, vol(0.1), 0.12);
      navigator.vibrate?.([20, 10, 20]);
    }, [vol, getCtx]),

    demonActivate: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;
      // 0.15s 無音 → 爆発
      setTimeout(() => {
        synth(ctx, 'sawtooth', 40, 40, 0.6, vol(0.9));
        noise(ctx, 0.3, vol(0.4), 300);
        synth(ctx, 'sine', 1320, 1320, 0.4, vol(0.15), 0.3);
        navigator.vibrate?.([40, 20, 60]);
      }, 150);
    }, [vol, getCtx]),

    demonEnd: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;
      synth(ctx, 'sine', 40, 20, 0.8, vol(0.3));

    }, [vol, getCtx]),

    gaugeFull: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;

        synth(ctx, 'sine', f, f, 0.3, vol(0.08), i * 0.07);
      });
      navigator.vibrate?.([20, 30, 20, 30]);
    }, [vol, getCtx]),

    jobChange: useCallback(() => {
      const ctx = getCtx(); if (!ctx) return;
      [262, 330, 392, 523].forEach((f, i) => {
        synth(ctx, 'sine', f, f, 0.15, vol(0.1), i * 0.1);
      });
      navigator.vibrate?.([20, 10, 30]);
    }, [vol, getCtx]),

    ssrDrop: useCallback(() => {

      }, 2000);
```

## 6. ストア設計

```typescript
// src/store/useAudioStore.ts
import { create } from 'zustand';
import { persist } from 'zustand/middleware';
import { AudioService } from '../services/AudioService';

interface AudioState {
  bgmVolume: number;      // 0〜1
  seVolume: number;       // 0〜1
  muted: boolean;
  bgmEnabled: boolean;
  seEnabled: boolean;

  setBgmVolume: (v: number) => void;
  setSeVolume: (v: number) => void;
  setMuted: (v: boolean) => void;
  toggleBgmEnabled: () => void;
  toggleSeEnabled: () => void;
}

export const useAudioStore = create<AudioState>()(
  persist(
    (set, get) => ({
      bgmVolume: 0.7,
      seVolume: 0.8,
      muted: false,
      bgmEnabled: true,
      seEnabled: true,

      setBgmVolume: (v) => {
        set({ bgmVolume: v });
        AudioService.setVolume(v, get().seVolume);
      },
      setSeVolume: (v) => {
        set({ seVolume: v });
        AudioService.setVolume(get().bgmVolume, v);
      },
      setMuted: (v) => {
        set({ muted: v });
        AudioService.setMuted(v);
      },
      toggleBgmEnabled: () =>
        set(s => {
          AudioService.setVolume(s.bgmEnabled ? 0 : s.bgmVolume, s.seVolume);
          return { bgmEnabled: !s.bgmEnabled };
        }),
      toggleSeEnabled: () => set(s => ({ seEnabled: !s.seEnabled })),
    }),
    { name: 'audio-store' }
  )
);
```

## 8. ブラウザ自動再生ポリシー対応

Web ブラウザは最初のユーザー操作前の音声再生をブロックする。以下の方針を取る:

```typescript
// src/app/page.tsx の最初のクリックイベントで AudioContext を resume する
useEffect(() => {
  const unlock = () => {
    AudioService.transitionTo('HOME_LOBBY', 1.5);
    document.removeEventListener('pointerdown', unlock);
  };
  document.addEventListener('pointerdown', unlock, { once: true });
}, []);
```

**初回タップ時**: HOME_LOBBY BGM がフェードインで開始。ユーザーは操作した覚えがなく「気づいたら音楽が流れていた」体験になる（原神方式）。
