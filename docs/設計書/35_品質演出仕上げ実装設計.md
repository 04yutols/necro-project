# 35_品質演出仕上げ実装設計

> 更新日: 2026-05-16  
> 対象: `docs/progress/CH1_TODO.md` Phase C「UX完成度 / 品質・演出の仕上げ」

## 1. 目的

第1章の主要フローを「動く」段階から「遊びの報酬感がある」段階へ引き上げる。対象は戦闘終了後の鑑定演出、音響基盤、武器/残滓の装備詳細、エリアマップの進行演出とする。

## 2. 実装方針

| 領域 | 実装 | 意図 |
|---|---|---|
| ドロップ鑑定 | `ResultScreen.tsx` に PixiJS の `DropPixiVfx` を追加 | SR/SSR/UR の紫光柱・粒子・怨念柱を CSS だけでなく Canvas レイヤーで表現する |
| サウンド | `AudioService.ts` / `useBGM.ts` / `useSoundEffects.ts` / `useAudioStore.ts` | 実音源未配置でも iOS ブラウザで破綻しない WebAudio 合成BGM/SEを提供する |
| バトルSE | `BattleCanvas.tsx` | 通常攻撃、属性スキル、魔神化、魔神技、WAVEクリアに SE を同期する |
| 鑑定SE | `ResultScreen.tsx` | SSR/UR 鑑定時に専用BGMへクロスフェードし、URは低域ノイズと歪んだ和音で禍々しさを出す |
| 武器詳細 | `LegionHub.tsx` | パッシブ説明に加えて R1/現在/次/最大ランクの効果プレビューを表示する |
| 残滓強化 | `LegionHub.tsx` | 強化後に Tier 上昇結果モーダルを表示し、Lv/SCORE/主能力の変化を即時確認できるようにする |
| エリア進行 | `AreaMap.tsx` | 新規解放ノードに霧解除オーバーレイとノード周辺の粒子を出し、Layer 2 の進行感を明示する |

## 3. 音響設計

`AudioService` はシングルトンとして WebAudio の `AudioContext` を共有する。iOS Safari の自動再生制限に合わせ、最初の `pointerdown/touchstart` で `unlock()` し、事前に要求された BGM シーンは `pendingScene` として保持する。

### BGM シーン

- `HOME_LOBBY`
- `MAP_EXPLORE`
- `BATTLE_NORMAL`
- `BATTLE_BOSS`
- `DEMON_OVERLAY`
- `RESULT_WIN`
- `RESULT_LOSE`
- `NECRO_LAB`
- `STORY_NEUTRAL`
- `STORY_EMOTIONAL`
- `SSR_REVEAL`
- `UR_REVEAL`

BGM は現時点では mp3/ogg ではなく、低域ドローン・ノイズ・三和音の合成ループで表現する。将来実音源を追加する場合も、`AudioService.transitionTo(scene)` の呼び出し側を変えずに内部実装を差し替えられる。

### SE

- 通常攻撃: 短いノイズ + 三角波
- 属性スキル: `ElementType × SkillAttackType` から周波数帯とノイズフィルタを選択
- 魔神化: `DEMON_OVERLAY` を開始し、低域ノイズと歪んだ和音を重ねる
- 魔神技: 長めの低域インパクト
- 鑑定: `COMMON/SR/SSR/UR` の4段階。URのみ `UR_REVEAL` へ即時遷移
- 残滓強化: 通常強化と Lv 上昇で音程/持続を変える

## 4. VFX設計

### 鑑定 PixiJS VFX

`DropPixiVfx` は `ResultScreen` 内に絶対配置され、`Application.init({ resizeTo: host })` で親要素へ追従する。

- SR/RARE: 紫系の細い光柱、少量の粒子
- SSR/LR: 金色の光柱、リング、粒子の収束
- UR/HIDDEN_UNIQUE: 紫 + 赤黒の柱、縦に伸びる怨念粒子、中心の黒い裂け目

CSS のカード演出は維持し、Canvas は背景VFXレイヤーとして使う。これによりカードUIの可読性を落とさず、レアリティ差の体感を強くする。

### 霧解除

`AreaMap` は `sessionStorage` に `clearedStages.length + nextStage.id` を記録し、同じ解放演出を同一セッションで繰り返しすぎないようにする。演出は Layer 2 エリアマップ上で表示する。

## 5. UI/UX仕様

### 武器詳細

武器詳細は以下を同一パネルで確認できる。

- 基礎ATK / 最終ATK / ATKボーナス / 装備差分
- パッシブA/Bの現在ランク説明
- R1 / 現在 / 次 / R5 のランク別プレビュー

### 残滓強化

強化ボタン押下後、投入EXPと更新後ステータスをモーダルで確認する。

- Level before/after
- SCORE before/after
- 主能力 before/after
- Tier上昇時は `TIER ASCENDED` 表示、王冠アイコン、強めの発光

## 7. 残課題

- 実音源アセットを導入した場合は `AudioService` 内部で合成BGMからファイルBGMへ差し替える。
- 設定画面に音量スライダーを露出する。
- SSR/UR の初回獲得時だけより長い専用カメラ演出を追加する。
