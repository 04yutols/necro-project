# 13 — 武器性能・数理設計書

> 最終更新: 2026-05-07  
> 実装参照: `src/logic/WeaponSystem.ts`, `src/components/legion/LegionHub.tsx`, `src/store/useGameStore.ts`, `docs/設計書/14_武器UIUX実装設計.md`

武器はキャラクターの**最終火力を決定する最大の基盤**。
本設計では、レアリティごとの「基礎ステータス格差」と「パッシブスキルの複雑さ」を数理的に定義し、
下位〜最上位（UR）までの明確なヒエラルキーと、逆転の可能性（ジャイアントキリング）を構築する。

---

## 1. 最終攻撃力の計算式

```
FinalATK = (CharacterBaseATK + WeaponBaseATK) × (1 + ATKBonus%) + FlatATK
```

| 変数 | 出所 |
|---|---|
| `CharacterBaseATK` | キャラクターの `stats.atk`（職業修正・パッシブ込み） |
| `WeaponBaseATK` | 武器の基礎攻撃力（レアリティ×アーキタイプで決定） |
| `ATKBonus%` | 武器サブステ・深淵の残滓・ジョブパッシブの合算 |
| `FlatATK` | 深淵の残滓サブステ「ATK実数」等の加算値 |

> **インフレの構造**: `WeaponBaseATK` の差は終盤ほど乗算バフにより増幅されるため、
> レアリティ格差が実質的に拡大する。これをコントロールするために「ステータス予算」を設ける。

---

## 2. ステータス予算と反比例の法則

**基礎攻撃力が高い型ほど、サブステータスの係数を低くする。**
これにより同レアリティ内での性能差を縮小し、プレイヤーの選択に意味を持たせる。

> **例外**: UR（神話）のみ、この法則を意図的に破壊する（§2.1参照）。

### 2.2 成長曲線の仕様

- **Lv1〜90**: 線形スケーリング
- **限界突破（上限解放）タイミング**: 固定ボーナス値が加算される **区分線形スケーリング**
  - 限界突破直後に基礎ATKがステップアップし、成長曲線が階段状になる

```
// 限界突破ボーナスの計算イメージ
weaponATK(lv, rank) =
  baseATK(lv) + breakBonus[rank]

// breakBonus (概算例・SR中ベース型)
//   rank 0 (無凸):    +0
//   rank 1 (1凸):    +15
//   rank 2 (2凸):    +30
//   ...
//   rank 5 (完凸):  +75
```

## 3. 固有スキル（パッシブ）の設計法則

すべての武器は以下2スロットのパッシブ効果を持ち、限界突破（魂の共鳴：ランクⅠ〜Ⅴ）で強化される。

| スロット | 名称 | 役割 |
|---|---|---|
| Effect A | **基礎昇華** | 常時発動するステータス底上げ（会心率・HP上限等） |
| Effect B | **深淵の理** | コアシステム（魔神化・行動値・霊魂砕き等）への特殊干渉 |

---

### 3.1 レアリティごとのパッシブ複雑度階層

レアリティが上がるほど「**発動条件の複雑さ**」と「**システムへの干渉度**」が増す。
単純な数値暴力ではなく、プレイヤーにロールを理解・活用させる設計とする。

#### R（凡庸）: 単一条件 + 単一効果

- **法則**: システム干渉なし。シンプルだが効果は控えめ
- **例**:
  > 敵を倒した時、自身のHPを5%回復する。

#### SR（業物）: 特定条件 + 強力な単一効果

- **法則**: 特定の属性・状態異常を要求するが、条件を満たせばSSR無凸を上回る火力（ジャイアントキリング）を発揮
- **例**:
  > 敵が「出血」状態の時、会心率が15%上昇し、行動値が10%早まる。

#### SSR（遺物）: スタック（蓄積）条件 + 複合システム干渉

- **法則**: 戦闘中の行動（ローテーション）を武器に合わせる必要があるが、見返りが絶大。「魔神化」「霊魂砕き」を直接ブーストする
- **例**:
  > スキル使用ごとに「残響」スタックを獲得（最大3）。  
  > スタック3消費時、追加の闇属性ダメージを与え、自身の「魔神化ソウルゲージ」を20%即時回復する。

#### UR（神話）: 無条件 + コア・ルールの破壊

- **法則**: 発動条件なしでシステムそのものを捻じ曲げるチート性能
- **例**:
  > 無条件で全ダメージ+30%。  
  > さらに、この武器の攻撃は敵の「霊的防壁（属性シールド）」を無視して貫通ダメージを与え、防御力を半減させる。

---

### 3.2 限界突破（魂の共鳴）のスケーリング法則

限界突破では**基礎攻撃力は一切変化せず**、パッシブスキルの効果量のみが強化される。

#### 原則: 2倍の法則

ランクⅠの効果量が、ランクⅤ（完凸）時点で**正確に2倍**に成長する。

```
例:
  ランクⅠ: 全ダメージ +15%
  ランクⅡ: 全ダメージ +19%
  ランクⅢ: 全ダメージ +22%
  ランクⅣ: 全ダメージ +26%
  ランクⅤ: 全ダメージ +30%  ← ランクⅠの2倍
```

中間値はベジェ曲線的な加速ではなく、**等差数列**（均等増加）を基本とする。

#### 例外: 確率・CDを持つサポート系武器の完凸保証

発動確率やクールタイム（CD）が設定されているサポート型武器は、完凸時に**運要素を完全排除**する。

| 項目 | ランクⅠ | ランクⅤ（完凸） |
|---|---|---|
| 発動確率 | 60% | **100%（確定発動）** |
| クールタイム | 12秒 | **6秒（極限短縮）** |

```
例:
  ランクⅠ: 会心時 60%の確率でエネルギーを回復 (CD 12秒)
  ランクⅤ: 会心時 100%の確率でエネルギーを回復 (CD 6秒)
```

## 5. ジャイアントキリングの成立条件

SRがSSRを上回るシナリオが成立するための必要条件を明文化する。

```
SR高ベース型（565）× SR低ベース型会心パッシブ（critRate +15%）× 完凸
  vs
SSR中ベース型（608）× 汎用パッシブ × 無凸

敵「出血」状態下での会心ダメージ期待値（仮計算）:
  SR: 565 × (1 + 0.08) × (165 + 15 + 11.6) / 100 ≈ 1,163
  SSR: 608 × (1 + 0.08) × 165 / 100              ≈  1,082

→ SR完凸が約7.5%上回る（条件付き）
```

設計意図: 課金武器（SSR/UR）を持つプレイヤーに絶対的優位を与えつつ、
特定の戦略（状態異常ビルド）では下位装備でも上位に勝てる可能性を残すことで
**課金しない層のモチベーション**と**コミュニティ内の戦術議論**を維持する。

## 6. 型定義への影響

既存の `ItemData`（`src/types/game.ts`）を武器パッシブに対応させる。
現行UIの装備対象は `type: 'WEAPON'` のみ。

```typescript
interface ItemData {
  id: string;
  name: string;
  type: 'WEAPON';
  rarity: 'R' | 'SR' | 'SSR' | 'UR';
  stats: Partial<BaseStats>;
  weaponRarity?: 'R' | 'SR' | 'SSR' | 'UR';
  archetype?: 'LOW' | 'MID' | 'HIGH' | 'MYTHIC';
  rank?: number;               // 0〜5（魂の共鳴ランク）
  ilv?: number;                // 1〜90（打ち直し）
  passiveA?: WeaponPassive;    // 基礎昇華スロット
  passiveB?: WeaponPassive;    // 深淵の理スロット

interface WeaponPassive {
  nameJa: string;
  descTemplate: string;        // "{value}%" のようなテンプレート
  values: number[];            // [rankⅠ値, ...rankⅤ値] 5要素
  condition?: string;          // "BLEED" | "SOUL_GAUGE_FULL" | null（無条件）
  systemTag?: 'DEMON_MODE' | 'SOUL_SHATTER' | 'ACTION_VALUE' | 'SHIELD_PIERCE';
}
```

## 6-B. エコシステム設計（ドロップ＆強化循環）

本システムは**課金ガチャ・スタミナ制限を一切持たない純ハクスラ設計**。
すべての武器と強化素材は「ストーリーダンジョン」からのみドロップし、
プレイヤーの「時間と試行回数」だけが経済を回す完全循環型エコシステムを構築する。

---

### 6-B.1 ドロップ確率とレアリティの立ち位置

スタミナ無制限であるため、SSR以上の確率は**意図的に絞り込む**。

| レアリティ | 呼称 | ドロップ率 | エコシステム上の立ち位置 |
|---|---|---|---|
| **R** | 凡庸 | 約85.0% | 大半を占めるハズレ枠。ただし分解で限界突破素材「**凡骨のイデア**」になるため何本落ちても無駄にならない |
| **SR** | 業物 | 約14.5% | 周回における日常的な「当たり」。数日周回で突破素材が揃い、中盤〜後半の主力として機能する |
| **SSR** | 遺物 | 約0.5% | 大当たり。ドロップ瞬間に強烈な画面演出が走る。使わない武器種でも分解で「**英雄のイデア**」になるため、喜びが絶対に損なわれない |
| **UR** | 神話 | **???（実質0.01%未満）** | 【**理外の呪装**】プレイヤーがそのダンジョンを何百周と狂ったように周回し、大量虐殺された魔物たちの怨念と絶望が特異点に達した時、極稀に禍々しい武器として物理領域に顕現する。UI上のドロップリストには**記載されない** |

> **UR の世界観補足**: プレイヤー自身が生み出す「怨念の結晶」。
> 理外の存在であるため確率の概念が適用されず、システムのルールを破壊するチート級パッシブを持つ。

---

### 6-B.2 武器の寿命を無限にする「2軸の強化システム」

「同じSSRを引かないと強くなれない」「序盤SSRがレベル不足でゴミになる」という
ハクスラの欠点を解消し、**お気に入りの武器（特に血塗られたUR）を一生使い続けられる**仕組みを提供する。

#### 強化軸① — パッシブ強化: 限界突破（魂の共鳴）

固有スキル（Effect A / B）の効果量を最大2倍に引き上げる（§3.2参照）。

> **核心設計**: 同名武器は**不要**。不要な武器をネクロラボで分解すると、
> レアリティに応じた汎用突破素材「**魂のイデア**」を獲得できる。

| レアリティ | 分解で得られる素材 | 1凸あたり消費数（概算） |
|---|---|---|
| R | 凡骨のイデア | — |
| SR | 業物のイデア | R素材 × 多数 |
| SSR | 英雄のイデア | SR素材 × 多数 |
| UR | 完凸固定（分解不可） | — |

```
運用例:
  使わないSSR大剣（英雄のイデア × 3）を砕く
    → お気に入りのSSR大鎌の魂の共鳴ランクⅠ → Ⅱへ
  = 別武器種のSSRドロップが、本命武器の強化リソースに直結する
  = ドロップのガッカリ感をゼロにする
```

#### 強化軸② — 基礎ステータス強化: 打ち直し（Reforge）

武器の基礎攻撃力（アイテムレベル = **ILv**）をプレイヤーの現在水準まで引き上げるシステム。
最新ダンジョンでのみドロップする専用素材「**深淵の黒鋼（Abyssal Obsidian）**」を消費する。

```
仕様例:
  序盤で拾った ILv.20 の SSR大鎌
    + 深淵の黒鋼（現在の適正ダンジョンでのみドロップ）を消費
    → ILv.80 の SSR大鎌に「打ち直し」
    ※ パッシブ・ランク・外見はすべて維持される
```

| 打ち直し段階 | 必要素材（概算） | 基礎ATK上昇（中ベースSSR例） |
|---|---|---|
| ILv.20 → 40 | 深淵の黒鋼 × 10 | 608 → 635 |
| ILv.40 → 60 | 深淵の黒鋼 × 25 | 635 → 657 |
| ILv.60 → 80 | 深淵の黒鋼 × 50 | 657 → 674 |
| ILv.80 → 90 | 深淵の黒鋼 × 100 + 英雄のイデア | 674 → 最大値 |

> **設計意図**: 最新ダンジョンでのみ素材がドロップするため、
> 古い武器を育て続けるには**常に最前線へ進む動機**が生まれる。

---

### 6-B.3 完全循環型ハクスラループ（殺戮の螺旋）

プレイヤーのすべての行動が無駄なく循環し、モチベーションを永続させる。

```
① 殺戮の螺旋
   スタミナなし → 目当てのSSR（あるいは怨念のUR）を求めて
   ストーリーダンジョンを高速周回し、魔物を狩り続ける

② 一喜一憂と分解
   大量に落ちたR・SR・不要なSSRを鑑定し、
   ネクロラボで一括分解 → 大量の「魂のイデア（突破素材）」へ変換

③ 確実な前進
   変換したイデアで本命武器を完凸
   最新ダンジョンの黒鋼で基礎ATKを最前線まで打ち直す

④ さらなる深淵へ
   強化された愛銃（呪装）を手に、次のダンジョンへ
   → ① に戻る
```

**ループの自己強化構造**:
- R・SR のドロップ → イデアに変換 → 本命武器を強化 → 難しいダンジョンに挑める → より良い黒鋼がドロップ
- 「ハズレ」が存在しない設計。すべての武器ドロップが強化の燃料として機能する
