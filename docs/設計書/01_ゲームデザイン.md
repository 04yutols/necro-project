## 1. ゲームコンセプト

| 項目 | 内容 |
|---|---|
| タイトル | ネクロマンス・ブレイブ |
| ジャンル | 死霊術ハック＆スラッシュRPG / 魔王育成ターン制RPG |
| ターゲット | 硬派なハクスラ・ビルド構築・超低確率ドロップに熱狂するプレイヤー |
| プラットフォーム | Web ブラウザ（PWA対応）、スマホ操作最適化 |
| デザイン | Gothic-Morphism（Void Purple #8B00FF / Obsidian Glass） |

### ストーリー
魔族の少年（主人公）には種族を超えた親友「勇者の少年」がいた。しかし親友は「現代の魔王」に殺され、死の間際に主人公へ「勇者の力」を託す。主人公は **「親友の復讐」** と **「新魔王になる」** という2つの目的のため、相反する2つの力を手に魔王への道を進む。

**第二部構想**: 新魔王になった後、天界や他の魔王候補との広範な戦いが始まる。

## 3. 職業システム（勇者の力）GDD-004

### 設計思想
転職しても「過去の経験（パッシブボーナス）」は消えない。どの職業でどれだけレベルを上げたかが、最終的な強さに積み上がる。

### 永続パッシブ蓄積
職業レベルが特定のマイルストーンに達すると、**転職後も消えない**ステータスボーナスを獲得する。

例（剣士）:
- Lv10: passiveAtkBonus +5
- Lv20: passiveAtkBonus +10
- Lv30: passiveAtkBonus +15, passiveDefBonus +5

### Energyリソース設計
| カテゴリ | 最大EN | スキルコスト | 戦略 |
|---|---|---|---|
| PHYSICAL（剣士・盗賊） | 標準〜高（90-120） | 低〜中 | 通常攻撃でENを戻しながら手数を出す |
| MAGICAL（魔法使い・ネクロ） | 低〜高（80-120） | 中〜高 | 属性スキルと奥義のタイミングを管理する |

## 5. アイテム体系 GDD-007

### 装備スロット（現行）
`WEAPON` + 深淵の残滓 5枠。

旧案の `SUB` / `HEAD` / `BODY` / `ARMS` / `LEGS` / `ACC1` / `ACC2` はDB互換フィールドとして残すが、
ゲーム設計上の防具・装飾品スロットは廃止。

### レアリティ
| レアリティ | 入手 | 特徴 |
|---|---|---|
| R | 通常ドロップ | 分解で凡骨のイデア |
| SR | 通常ドロップ | 条件付きでSSR無凸を越える業物 |
| SSR | 低確率ドロップ | 強力なパッシブと高い基礎ATK |
| UR | 怨念特異点 | 理外の呪装。分解不可・演出強化対象 |

### 深淵の残滓（AbyssalResidue）
- Genshin Impact の「聖遺物」ライクなビルドアイテム
- メインステータス（固定）+ サブオプション（ランダム4種）
- レベル1〜20まで強化可能（素材消費）
- レアリティ: COMMON / RARE / EPIC

### 霊核（SpiritCore）
- モンスターの属性付与・スキル変更・ATK倍率変更
- 敵ドロップまたは条件解放で入手

## 6. 第一発見者システム（GDD追加要件）

世界で最初にそのアイテムをドロップしたプレイヤーの名前とシリアルNoがアイテムに永久刻印される。

```
Item.isUnique     → true (HIDDEN_UNIQUE判定)
Item.discovererId → User.id
Item.discovererName → User.name
Item.serialNo     → グローバルシーケンス（発見順）
Item.discoveredAt → 発見日時
```

**「鑑定証明書」**: 古文書風デザインの画像を Canvas で生成し、SNS共有可能。

## 7. 収益化方針

| カテゴリ | 内容 |
|---|---|
| QoL課金 | インベントリ・倉庫拡張、自動フィルタリング |
| サポーターパス | 月額制、EXP微増、専用称号 |
| エフェクト課金 | 攻撃エフェクト・召喚獣スキン変更 |

**設計方針**: 闇術師スキル（回復・状態異常解除）の価値を損なわないよう、消費アイテムは効果が低いか高価に設定。**EN回復アイテム（エーテル等）は非売品**（宝箱・レアドロップのみ）。
