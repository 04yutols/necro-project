# 07 — デザインシステム（Gothic-Morphism）

> 最終更新: 2026-05-06  
> 実装ファイル: `src/app/globals.css`, `tailwind.config.ts`

## 1. デザインコンセプト

**Gothic-Morphism** = Honkai Star Rail × Genshin Impact × ゴシック建築

- **Obsidian Glass**: 半透明の黒曜石ガラス（背景 + glassmorphism）
- **Void Purple**: 魔力を象徴する深紫ネオン（#8B00FF）
- **Cursed Gold**: 古びた金（#D4AF37）でレアリティ・選択状態を表現
- **Bone Frames**: ゴシック建築の骨格を UI ボーダーに見立てる
- **Noise Grain**: 微細なグレインテクスチャで「古さ」を演出

### Framer Motion 標準パターン

```tsx
// スクリーン スライドイン（⚠️ overflow:hidden は分離すること）
<motion.div
  initial={{ y: '100%' }} animate={{ y: 0 }} exit={{ y: '100%' }}
  transition={{ type: 'spring', stiffness: 295, damping: 33 }}
  style={{ position: 'absolute', inset: 0 }}  // transformのみ
>
  <div className="absolute inset-0 flex flex-col overflow-hidden">  // overflowはここ

// フェードイン
initial={{ opacity: 0, y: 8 }} animate={{ opacity: 1, y: 0 }}
transition={{ duration: 0.22, ease: 'easeOut' }}

// キャラクター浮遊
animate={{ y: [-6, 6, -6] }}
transition={{ duration: 3.8, repeat: Infinity, ease: 'easeInOut' }}

// タップフィードバック
whileTap={{ scale: 0.96 }}
```

## 10. Demon Mode カラーオーバーライド

`isDemonMode == true` のとき、LegionHub は `data-demon="true"` 属性を最上位 div に付与する。

```css
/* globals.css */
[data-demon="true"] {
  --demon-accent: #CC2222;
  --demon-border: rgba(139,0,0,0.4);
  /* 各コンポーネントは var(--demon-accent) を参照してカラー変更 */
}
```

効果:
- アリーナ背景が赤くなる
- ボーダーが赤くなる（`#dc2626`）
- ダメージテキストが赤（`#dc2626`）
- 「魔神化 ON」バッジがパルスグロー

## 11. Tailwind v4 カスタムトークン運用

`tailwind.config.ts` の `designTokens` オブジェクトで一元管理。

```typescript
// tailwind.config.ts
const designTokens = {
  colors: {
    void: { DEFAULT: '#8B00FF', bright: '#BC00FB', ... },
    secondary: { DEFAULT: '#D4AF37', ... },
    // ...
  }
};

// コンポーネントで使用
className="text-void border-void/30 bg-void/10"
className="text-secondary border-secondary/40"
```

CSS 変数との使い分け:
- `tailwind.config.ts` → Tailwind ユーティリティクラスとして使用したいトークン
- `globals.css` `--xxx` → ガラス効果・グロー shadow など複雑な値
