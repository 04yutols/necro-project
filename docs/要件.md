『ネクロマンスブレイブ』完全再構築・要件定義書 (REBOOT 2026)
1. 開発ミッション
既存の全コードを破棄し、**「ナイト・アンド・ドラゴン（K&D）の機能性」と「死霊術師のダークな世界観」**を融合させたハクスラRPGを0から構築せよ。

2. 視覚的・構造的アイデンティティ（Dark Ancient Codex）
UIコンセプト: 「闇の魔導書（グリモワール）」そのもの。

素材感: 黒耀石の革、チャコールグレーの皮膚、腐食した黒鉄、骨の装飾。

色彩: 基調は #0D0D0D（漆黒）、アクセントに #8A2BE2（ヴォイド・パープル）の脈動する光。

レイアウト (Mobile First): スマホ縦画面を3段構成で固定。

上段 (Visual Area): PixiJSによる2.5D戦闘描画 or キャライラスト。

中段 (Narrative Log): 古文書風の枠で囲まれた、テキストベースの戦闘・イベントログ。

下段 (Command Panel): 骨の装飾が施された、タクティカルなコマンドボタン群。

3. 技術スタック
Frontend: Next.js (App Router), Tailwind CSS

Graphics: PixiJS (エフェクト、バトル演出、ドロップ儀式)

Database: PostgreSQL (Prisma/Supabase)

JSONB を活用し、未定義のランダムオプションを柔軟に格納する。

4. コアゲームシステム（K&Dスタイルの正統進化）
4.1 ハクスラ・アイテム体系
武器・防具 (Base Equipment):

基礎性能固定。使用によりレベルアップする「魂の共鳴（成長）」要素。

深淵の残滓 (Abyssal Remnants):

厳選用ビルドアイテム。ドロップ時にメイン1＋サブ4のステータスをJSONBでランダム生成。

霊核 (Soul Cores):

召喚獣のアンロックと魔改造用。

4.2 ドロップ・演出（脳汁要素）
紫の柱 (The Ritual): - 隠しユニーク（ドロップ率 0.01%）当選時、リザルト画面でBGMを停止させ、PixiJSで画面を突き破る「紫の光柱」を演出。

歴史の刻印:

全プレイヤー共通の discovery_log テーブルを参照。

第一発見者の名前とシリアルNo（例：No.0001）をアイテムに永続的に刻印。

5. 実装指示（AIエージェントへの段階的命令）
Step 1: データベース・スキーマ設計
items (JSONB含む), user_status, summons, discovery_log のテーブル定義を、リレーションを考慮して作成せよ。

Step 2: グローバル・テーマの設定
Tailwind CSSを拡張し、void-purple や bone-white などの独自カラー、および「闇の古文書」の質感を出すためのカスタムユーティリティを設定せよ。

Step 3: レイアウト・コンポーネントの構築
前述の3段構成（Visual / Log / Command）を持つメインレイアウトを実装せよ。

Step 4: K&D風バトルエンジンの実装
ターン制、WAVE制のロジックを構築。行動順（SPD依存）に基づいたコマンド選択、および中段ログへのテキスト出力機能を実装せよ。

Step 5: 鑑定・シェア機能の構築
リザルト画面でのタップ鑑定演出、およびPixiJSを用いた「古文書風鑑定証明書」の画像生成ロジックを実装せよ。
