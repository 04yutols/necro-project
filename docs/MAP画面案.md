# Role: Senior Game Developer (Next.js, PixiJS, Tailwind CSS)
# Task: Implement "Necromance Brave" Map System

提供された「闇の魔導書に描かれた古地図」の画像を背景に、ハクスラRPG『ネクロマンスブレイブ』のMAP選択画面をスクラッチから構築せよ。

## 1. デザイン要件
- 素材感: 画像の「セピア色の羊皮紙」と「不気味な紫の魔力」をデザインの基準とする。#ワールドマップ.pngを使用すること
- UI構成: スマホ縦画面 3段構成。
  - [cite_start]上段: ヘッダー（エリア名、資金、龍銀 [cite: 18, 19]）。
  - 中段: マップ表示エリア（画像アセットを背景に、PixiJSでインタラクティブなノードを描画）。
  - 下段: ステージ詳細パネル（K&D風の出撃確認画面）。

## 2. ロジック要件 (Knight & Dragon Style)
- ノード選択: 各地点（1-1, 1-2等）をタップすると、下段パネルに情報を表示。
- 進行管理: 前のステージをクリアすると、次のステージへの「鎖（パス）」が解禁されるロジック。
- [cite_start]パーティ連携: 下段パネルには、現在のパーティの生存状態（HP/MPバー [cite: 27, 28]）を小さく表示。

## 3. 実装の優先順位
1. 地図画像の中央配置と、レスポンシブなスクロール/ズームの実装。
2. JSONデータから座標（x, y）を読み込み、地図上にノードをプロットする関数の作成。
3. ノード選択時のステート管理と、下段パネルのアニメーション表示。
4. 紫色の魔力（PixiJSパーティクル）による「攻略中ノード」の強調演出。
