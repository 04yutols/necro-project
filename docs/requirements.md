# GDD-001：ゲームコンセプトとストーリー
## 1. タイトル（仮）: ネクロマンスブレイブ
## 2. ジャンル: 魔王育成・ターン制RPG
## 3. ストーリー概要:
主人公: 魔族の少年。一族の悲願である「魔王になる」という夢を持つ。
発端: 主人公には種族を超えた親友「勇者の少年」がいた。しかし、その親友は「現代の魔王」によって殺害されてしまう。
力の継承: 親友は死の間際、主人公に自らの「勇者の力」を託す。
目的: 主人公は、**「親友の復讐（現代の魔王討伐）」と、自身の「魔王になる」**という2つの目的を叶えるため、2つの相反する力を手に、魔王への道を進む。
## 4. 拡張性（第二部）:
第一部 完: 「現代の魔王」を討伐し、復讐を果たす。主人公は正式に「新魔王」となる。
第二部（エンドコンテンツ）: 「新魔王」となった主人公を討伐対象と見なす「天界」や、他の「魔王候補」との、より広範な戦いが始まる。
# GDD-002：コア・ゲームループと構造
## 1. ゲームループ: プレイヤーは以下のサイクルを繰り返す。
拠点（街）: 準備・育成。
ワールドマップ: 挑戦する「エリア」を選択。
エリアマップ: 挑戦する「ステージ」を選択。
ステージ（戦闘）: WAVE制のターン制バトル（3WAVE制など）。
リザルト: 経験値、ゴールド、アイテム、「魂（使役モンスター）」を獲得。
拠点（街）: 獲得したリソースで「職業」と「死霊術」を強化する。
## 2. マップ構造:
ワールドマップ（「始まりの平原」「灼熱の火山」など、複数のエリアが並ぶ）
エリアマップ（「平原 1-1」「平原 1-2」「[ボス]ゴブリンの砦」など、複数のステージが並ぶ）
ステージ（WAVE制の戦闘）
各エリアの最終ステージには、強力な「エリアボス」が存在し、これを倒すと次の「エリア」が解放される。
# GDD-003：プレイヤーシステム（ステータスとリソース）
## 1. 9種のステータス:
HP (生命力)
MP (魔力)
ATK (物理攻撃力)
DEF (物理防御力)
MATK (魔法攻撃力)
MDEF (魔法防御力)
AGI (素早さ): 行動順、回避率
LUCK (運): 会心率、状態異常成功/抵抗率、ドロップ率
TEC (技巧): 会心威力、ダメージ安定性
## 2. リソース（MP）のバランス設計:
リソースは「MP」に統一する。（TP制は不採用）
物理系職業 (剣士, 盗賊):
最大MPは非常に低い（LvUPで最低限しか伸びない）。
スキル消費MPは**「極端に低い」**（例: 渾身斬り MP 3）。
「低コストスキルを連発できる回数」で戦う。
魔法系職業 (魔法使い, 闇術師):
最大MPは非常に高い（LvUPで爆発的に伸びる）。
スキル消費MPは**「高い」**（例: ファイア MP 15）。
「高コストスキルをWAVE戦でどう配分するか」というリソース管理で戦う。
# GDD-004：職業システム（勇者の力）
## 1. 拠点施設:
影の儀式場 (Umbral Rite-Hall): 主人公が異質な力（勇者）を解析・制御・変質させる場所。
## 2. 育成システム:
レベル: 職業ごとに独立（一次職 Lv30, 二次職 Lv40など）。転職するとLv1に戻る。
永続パッシブ: 職業レベルが一定に達すると、全職業に永続蓄積されるステータスボーナス（ATK+5 物理ダメージ+2%など）を獲得する。
## 3. 職業一覧:
Tier 1 (一次職):
剣士 (Warrior)
魔法使い (Mage)
闇術師 (Dark Priest): 主人公の魔力で「聖」が「闇」に変質。攻撃（ドレイン）が回復を兼ねる。
盗賊 (Rogue)
Tier 2 (二次職):
魔剣士 (Dark Knight) (剣士 + 闇術師)
ベルセルク (Berserker) (剣士 + 盗賊)
アークメイジ (Archmage) (魔法使い + 闇術師)
ソーサラー (Sorcerer) (魔法使い + 盗賊)
ウォーロック (Warlock) (闇術師 + 魔法使い)
ネクロマンサー (Necromancer) (闇術師 + 剣士): 死霊術システムを直接強化する唯一の職業。
アサシン (Assassin) (盗賊 + 剣士)
トリックスター (Trickster) (盗賊 + 魔法使い)
# GDD-005：死霊術システム（魔族の力）
## 1. 拠点施設:
死霊術研究所 (Necro-Lab)
## 2. 育成システム:
職業とは完全に独立した育成軸。
レベル (Lv.99): サクサク上がる。MaxCostと基礎ステータスが「少し」上昇。
ランク (Rank 10): Lv.99 到達後に「試練」をクリアするとRank UP。Lv.1に戻るが、MaxCostと基礎ステータスが**「爆発的に」**上昇する（転生システム）。
## 3. 軍団編成（3枠コスト制）:
戦闘に参加できる使役モンスターは3体固定。
プレイヤーは「MaxCost（死霊術ランクで上昇）」の範囲内で、コストの異なるモンスター（雑魚: コスト1〜, ボス: コスト3〜）を3枠に編成する。
## 4. カスタマイズ:
魂石化: 使役したモンスターを「ストック」から削除し、**「魂の欠片 (Soul Shard)」**という装備アイテムに変換する。
欠片装備: 「魂の欠片」を、他の使役モンスターに「装備」させ、ステータスや能力をカスタマイズする。
シナジー: [アンデッド] x 3 など、特定の種族で編成すると強力なパッシブバフが発動する。
## 5. 職業との連携:
プレイヤーの「現在の職業」に応じて、全使役モンスターにパッシブバフがかかる（例: 剣士なら軍団のDEF+15%）。
プレイヤーが**「覚醒」すると、このパッシブバフが超強化**される。
# GDD-006：戦闘とエリアギミック
## 1. 敵のランクとコスト:
ゲームの進行度（エリア）に応じて、敵の「ランク (Rank 1〜)」が上昇する。
敵の「死霊術コスト」は、この「ランク」と連動する（エリア10の雑魚(Rank 4 / コスト4) > エリア1のボス(Rank 1 / コスト3)）。
## 2. 学習曲線（エリアギミック）:
エリア1 (始まりの平原): フェーズ変化（ボスの覚醒）を学ぶ。
エリア2 (瘴気の沼): 状態異常への対処と、防御の使い分け（物理/魔法）を学ぶ。
エリア3 (鉄錆の要塞): 優先順位づけ（ヒーラー/バッファーから倒す）と複数部位のボスを学ぶ。
エリア4 (灼熱の火山): スリップダメージ環境下での、複合ギミック（上記全て）への対処を学ぶ。
GDD-007：アイテムと装備
## 1. 装備品:
スロット: 8枠（武器, サブ, 頭, 胴, 手, 足, アクセサリ1, アクセサリ2）
入手: 拠点の「店」での購入、およびボスからの「ユニーク装備」ドロップ（特殊効果付き）。
## 2. 消費アイテム:
バランス: 闇術師のスキル（回復・状態異常解除）の価値を損なわないよう、店売りアイテムは効果が低いか高価に設定する。
重要ルール: **MP回復アイテム（エーテル等）は「非売品」**とし、宝箱やレアドロップのみで入手可能とする。
