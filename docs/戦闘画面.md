1. 共通レイアウト構成
画面を縦に大きく3つのセクションに分割します。

【上段：35%】ビジュアル・タクティカル・エリア

2.5Dの戦闘フィールド。

最上部に「タイムライン（行動順バー）」。

【中段：25%】古文書ログ・エリア

戦闘状況が流れるテキストウィンドウ。背景は「闇の魔導書」のページ。

【下段：40%】コマンド・コンソール（操作域）

親指で操作しやすい「リングメニュー」と補助ボタン。

2. パターンA：【通常モード】（待機・基本）
親指の可動域に合わせて、右下に「魔方陣リング」を配置します。
